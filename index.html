GANHide: GAN-Based Steganography System

A deep learning–powered steganography system that hides binary messages within 8×8 grayscale digit images using conditional GANs (cGAN). The generated stego-images are visually indistinguishable from original inputs, achieving 50% discriminator accuracy—ideal for stealth. A custom CNN built with TensorFlow recovers hidden data with 97% training and 98% validation accuracy. The system also includes automated visual and statistical checks to ensure imperceptibility and message robustness across varying conditions.
